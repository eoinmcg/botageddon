# 🤖 ROBOCIDE

Twin stick shooter arcade action.

The not too distant future, the year is 1984. Humanity's
worst fear has come to pass: The machines have taken over.
A bot army is hell-bent on destroying all meat bags.
Only you stand between freedom and certain doom!
Buckle up soldier, we're in for some mayhem!

Controls:
keyboard & mouse
or use a gamepad
[M] mutes music & sfx
[P] pause
[O] fullscreen

#### [🎮 Play](https://eoinmcgrath.com/botageddon)

## Tech

Clone the repo, crack open a terminal and prepare for fun!

__Run locally__: `npm run dev`

__Create a zip__: `npm run zip`

__* Host on github pages:__ `npm run deploy`

(* In your repo under settings > pages)
